About This Site

Welcome to Bitwise, a minimalist, accessible blog showcasing projects and thoughts on technology. This site was built with Jekyll, focusing on accessibility, performance, and beautiful typography.

Features

  • Light and Dark Theme: Switch between themes with the toggle in the header.
  • Responsive Design: The site works beautifully on all screen sizes.
  • Accessibility Focused: High contrast colors and proper semantic HTML.
  • Custom Typography: Using the elegant Serifa font family.
  • Math Support: Beautiful mathematical equations with MathJax.
  • Code Highlighting: Syntax highlighting for code blocks.
